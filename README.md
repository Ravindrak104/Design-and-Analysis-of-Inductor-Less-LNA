# Design and Analysis of an Inductor-Less CMOS Low Noise Amplifier for 3.5 GHz 5G Applications

## Project Overview

This project presents the design and analysis of an **Inductor-Less CMOS Low Noise Amplifier (LNA)** operating at **3.5 GHz** for **5G wireless communication applications**. The circuit is designed and simulated using **Cadence Virtuoso** with **SCL 180 nm CMOS Technology**.

The objective of this work is to develop a compact and low-power LNA architecture by eliminating bulky on-chip inductors while maintaining acceptable RF performance. Various analyses including DC, AC, S-Parameter, Noise Figure, and Stability simulations were carried out to evaluate the amplifier.

---

## Objectives

- Design an Inductor-Less CMOS Low Noise Amplifier.
- Target operation at **3.5 GHz** for 5G applications.
- Reduce chip area by eliminating on-chip inductors.
- Achieve high gain with low power consumption.
- Analyze RF performance using Cadence Virtuoso.
